# 问题修复报告

## 最近修复 (v1.0.0)

### 1. 设置页面白屏
- **问题**: photos字段JSON字符串未解析导致崩溃
- **修复**: StoreSettingsPage load函数中解析photos字段

### 2. 登录失败提示不明确
- **问题**: 密码错误显示"请求失败"
- **修复**: 返回具体错误信息(密码错误/账号不存在)

### 3. 日志显示原始JSON
- **问题**: 修改记账日志显示未解析的JSON
- **修复**: 解析JSON后以原/改对比格式展示

### 4. OCR识别超时
- **问题**: 健康证OCR识别卡死在"正在识别"
- **修复**: 添加60秒超时机制

### 5. 通知缺少操作人
- **问题**: 通知不显示谁执行了操作
- **修复**: 通知内容添加[操作人]前缀

### 6. 盘点编辑后差异不消失
- **问题**: 手动编辑数量后差异仍显示
- **修复**: 编辑后自动清除该条目的差异显示

### 7. 角色权限完善
- **问题**: 店长/店铺管理员权限不一致
- **修复**: 统一权限配置，店长可添加/编辑员工

## 已知限制

1. **OCR精度**: Tesseract.js中文识别精度有限，部分字符可能识别错误
2. **图片压缩**: 大图片上传时可能较慢
3. **离线使用**: PWA离线功能有限，需要网络连接
