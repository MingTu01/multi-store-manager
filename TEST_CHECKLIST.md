## 二、门店管理（管理员）

### 2.1 创建门店
- [ ] 点击"新建门店"
- [ ] 不填名称直接保存 → 提示"请输入门店名称"
- [ ] 不添加股东直接保存 → 提示"请至少添加一位股东"
- [ ] 添加 1 个股东，比例 50% → 提示"股东份额相加必须等于 100%"
- [ ] 添加 2 个股东，比例 60%+40%（合计100%）→ 保存成功
- [ ] 添加门店照片（拍照/上传）→ 照片显示正常
- [ ] 保存后门店列表中出现新门店

### 2.2 编辑门店
- [ ] 点击门店卡片进入编辑
- [ ] 修改门店名称 → 保存成功
- [ ] 上传/删除门店照片 → 正常
- [ ] 修改股东信息 → 保存成功

### 2.3 删除门店
- [ ] 点击删除，不输入密码 → 提示"请输入管理员密码"
- [ ] 输入错误密码 → 提示"密码错误"
- [ ] 输入正确密码 → 门店删除成功
