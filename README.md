# Multi Shop Link

多店管理系统 - 适用于多门店经营的综合管理平台

版本：v1.1.6

## 技术栈

- 后端：Express 5 + TypeScript + SQLite（WAL模式）
- 前端：Vite + React 19 + TypeScript + Tailwind CSS 4
- PWA：支持桌面端和移动端安装
- OCR：Tesseract.js（离线中文识别）
- 端口：3001（可通过 PORT 环境变量修改）

## 自动构建流程

推送到源码仓库 `main` 分支 → GitHub Actions 自动构建前端 → 推送到部署仓库

## 部署方式

### 方式一：在线更新（推荐）
已部署的系统可通过 Web 端一键更新：
1. 管理员登录 → 系统设置 → 系统升级
2. 点击「检查更新」
3. 发现新版本后点击「执行更新」
4. 系统自动备份数据库、下载更新、覆盖文件、重启

### 方式二：Docker 部署
```bash
# 克隆部署仓库
git clone https://github.com/MingTu01/multi-shop-link-deploy.git
cd multi-shop-link-deploy

# 启动
docker-compose up -d
```

### 方式三：直接部署
```bash
# 克隆部署仓库
git clone https://github.com/MingTu01/multi-shop-link-deploy.git
cd multi-shop-link-deploy

# 安装依赖
npm install

# 启动
node --import tsx src/index.ts
```

## 功能模块

### 管理端（系统管理员）
- **管理大屏** — 全店收支总览、环比同比、收入支出构成饼图、门店概览卡片、趋势对比图
- **门店管理** — 创建/编辑/删除门店、照片上传、股东信息、初始资金
- **系统设置** — 数据备份恢复、在线更新、ZIP升级、消息推送配置、权限说明
- **账户管理** — 修改个人信息、密码、退出登录

### 店铺端
- **门店总览** — 快捷操作（记账/盘点/开闭店/报表）、今日/本周/本月/本年/总数据
- **记账** — 收入/支出记录、分类管理、日/周/月/年/总筛选、日期选择器
- **盘点** — 添加条目（含拍照）、消耗记录、拖动排序、状态标记
- **开闭店** — 霸屏模式、拍照强制、交接内容、历史记录
- **报表** — 收入支出构成饼图、环比同比对比、趋势图、资金余额
- **工资** — 生成工资条、确认发放、个人工资单弹窗
- **分红** — 按股份比例自动计算、确认分红
- **员工管理** — 添加/编辑/删除员工、头像上传、岗位/月薪/角色/状态
- **日志** — 操作日志记录、筛选搜索
- **消息通知** — 通知角标、已读/未读
- **消息推送** — 店铺级推送配置（PushPlus/Server酱/企业微信）
- **我的** — 账户信息、修改密码、健康证上传（OCR识别）

### 通知系统
- 每日简报（21:00）、每周报告（周一）、每月报告（1号）
- 健康证到期提醒、工资确认、分红归档、开闭店通知
- 推送渠道：PushPlus、Server酱、企业微信自建应用
