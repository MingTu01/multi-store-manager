# Multi Shop Link (Multi Shop Link)

> 适用于Multi Store Management的综合管理平台，支持桌面端和移动端PWA

## 版本：v0.5.1

## 技术栈

- **后端**: Express 5 + TypeScript + SQLite (WAL模式)
- **前端**: Vite + React 19 + TypeScript + Tailwind CSS 4
- **OCR**: Tesseract.js (离线中文识别)
- **拖动排序**: dnd-kit
- **默认端口**: 5173 (可通过 PORT 环境变量修改)

## 功能模块

### 管理端 (ADMIN)
- **管理大屏** — 全店收支总览、环比同比、收入支出构成饼图、门店对比
- **门店管理** — 创建/编辑/删除门店、多张照片、股东信息
- **系统设置** — 数据备份恢复、ZIP升级、消息推送配置
- **账户管理** — 修改个人信息、密码、退出登录

### 店铺端
- **门店总览** — 快捷操作(记账/盘点/开闭店/报表)、今日/本月/本年数据
- **记账** — 收入/支出记录、分类管理、日/周/月/年/总筛选
- **盘点** — 添加条目(含拍照)、领出功能、拖动排序、状态标记
- **开闭店** — 霸屏模式、拍照强制、交接内容
- **报表** — 收入支出构成饼图、环比同比对比、趋势图
- **工资** — 生成工资条、确认发放、个人工资单弹窗
- **分红** — 按股份比例自动计算、确认分红
- **员工管理** — 添加/编辑/删除员工、头像上传、岗位/月资/角色/状态、员工详情弹窗(含健康证)
- **健康证** — OCR识别、有效期管理、到期提前通知
- **设置** — 门店信息编辑、分类管理、消息推送配置

### 消息通知系统
- **站内通知** — 通知角标、已读/未读、48小时自动清除
- **通知类型** — 记账/工资/分红/盘点/员工/健康证/门店/开闭店
- **操作人显示** — 通知显示操作人名字
- **店铺标签** — 通知显示关联店铺名称

## 安装部署

```bash
# 克隆项目
git clone https://github.com/MingTu01/multi-shop-link.git
cd multi-shop-link

# 安装依赖
cd apps/server && npm install
cd ../web && npm install

# 构建前端
cd apps/web && npx vite build

# 启动服务
cd apps/server && node --import tsx src/index.ts
```

## 数据备份

- 备份文件为 .zip 或 .db 格式
- 通过"系统设置" → "上传备份" 恢复数据
- 建议定期备份 `apps/server/data/store.db`

## 升级方式

1. 在"系统设置"页面上传ZIP升级包
2. 或手动拉取最新代码后重新构建
