# Multi Shop Link

多店管理系统 - 适用于多门店经营的综合管理平台

版本：v1.1.6

## 技术栈

- 后端：Express 5 + TypeScript + SQLite（WAL模式）
- 前端：Vite + React 19 + TypeScript + Tailwind CSS 4
- PWA：支持桌面端和移动端安装
- OCR：Tesseract.js（离线中文识别）
- 拖动排序：@dnd-kit
- 端口：3001（可通过 PORT 环境变量修改）

## 功能模块

### 管理端（系统管理员）
- **管理大屏** — 全店收支总览、环比同比、收入支出构成饼图、门店概览卡片
- **门店管理** — 创建/编辑/删除门店、照片上传、股东信息、初始资金
- **系统设置** — 数据备份恢复、ZIP升级、消息推送配置、权限说明
- **账户管理** — 修改个人信息、密码、退出登录

### 店铺端
- **门店总览** — 快捷操作（记账/盘点/开闭店/报表）、今日/本月/本年数据
- **记账** — 收入/支出记录、分类管理、日/周/月/年/总筛选、日期选择器
- **盘点** — 添加条目（含拍照）、领出功能、拖动排序、状态标记（正常/差异/丢失/报废/待补货）
- **开闭店** — 霸屏模式、拍照强制、交接内容、历史记录
- **报表** — 收入支出构成饼图、环比同比对比、趋势图、资金余额
- **工资** — 生成工资条、确认发放、个人工资单弹窗（支持键盘左右/滑动切换）
- **分红** — 按股份比例自动计算、确认分红、个人分红单弹窗
- **员工管理** — 添加/编辑/删除员工、头像上传、岗位/月薪/角色/状态
- **日志** — 操作日志记录、筛选搜索
- **设置** — 门店基本信息、收支分类管理
- **消息通知** — 通知角标、已读/未读、48小时自动清除
- **消息推送** — 店铺级推送配置（PushPlus/Server酱/企业微信）
- **我的** — 账户信息、修改密码、健康证上传（OCR识别）

### 通知系统
- **每日简报** — 每日21:00推送经营汇总
- **每周报告** — 每周一推送周报
- **每月报告** — 每月1号推送月报
- **健康证到期** — 提前30天通知，每周一次，最后7天每天通知
- **工资确认** — 工资发放通知
- **分红归档** — 分红确认通知
- **开闭店** — 开店/闭店通知

### 推送渠道
- PushPlus（微信推送）
- Server酱（微信推送）
- 企业微信自建应用（支持图文消息）

## 部署

### 环境要求
- Node.js 18+
- npm 9+

### 安装步骤
```bash
# 1. 解压升级包
unzip multi-shop-link-v1.1.6.zip
cd multi-shop-link

# 2. 安装依赖
npm install

# 3. 启动服务
node --import tsx src/index.ts
```

### 默认账户
- 管理员：admin / 123456

### 升级方法
1. 在系统设置页面上传 ZIP 升级包
2. 系统自动验证、备份、升级
3. 点击确认刷新页面
