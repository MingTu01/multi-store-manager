# Multi Shop Link

多店管理系统 — 适用于多门店经营的综合管理平台，支持桌面端和移动端 PWA，iOS 原生 UI 风格。

**当前版本：v1.1.17**

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Express 5 + TypeScript + SQLite (WAL) |
| 前端 | Vite + React 19 + TypeScript + Tailwind CSS 4 |
| OCR | Tesseract.js（离线中文识别） |
| 拖动排序 | @dnd-kit |
| 端口 | 3001（可通过 PORT 环境变量修改） |

## 快速开始

### 本地开发

```bash
# 克隆源码仓库
git clone https://github.com/MingTu01/multi-store-manager.git
cd multi-store-manager

# 安装后端依赖
cd apps/server
npm install

# 启动后端
node --import tsx src/index.ts

# 新终端 - 安装前端依赖
cd apps/web
npm install

# 启动前端开发服务器
npx vite
```

## 功能模块

### 管理端（系统管理员）
- **管理大屏** — 全店收支总览、环比同比、收入支出构成饼图、门店概览卡片、趋势对比图
- **门店管理** — 创建/编辑/删除门店、照片上传、股东信息、初始资金
- **系统设置** — 数据备份恢复、在线更新、ZIP 升级、消息推送配置、权限说明
- **账户管理** — 修改个人信息、密码、退出登录

### 店铺端
- **门店总览** — 快捷操作（记账/盘点/开闭店/报表）、今日/本周/本月/今年/总数据
- **记账** — 收入/支出记录、分类管理、日/周/月/年/总筛选、日期选择器
- **盘点** — 添加条目（含拍照）、消耗记录、拖动排序、状态标记
- **开闭店** — 霸屏模式、拍照强制、交接内容、历史记录
- **报表** — 收入支出构成饼图、环比同比对比、趋势图、资金余额
- **工资** — 生成工资条、确认发放、个人工资单弹窗
- **分红** — 按股份比例自动计算、确认分红
- **员工管理** — 添加/编辑/删除员工、头像上传、岗位/月薪/角色/状态
- **日志** — 操作日志记录、筛选搜索
- **消息通知** — 通知角标、已读/未读
- **消息推送** — 店铺级推送配置（PushPlus/Server酱/企业微信）
- **我的** — 账户信息、修改密码、健康证上传（OCR 识别）

### 通知系统
- 每日简报（21:00）、每周报告（周一）、每月报告（1号）
- 健康证到期提醒、工资确认、分红归档、开闭店通知
- 推送渠道：PushPlus、Server酱、企业微信自建应用

## 开发规范

- 禁止使用 PowerShell 写文件（编码问题）
- 编辑和修改代码文件必须使用 Node.js
- 打包 ZIP 必须使用 Node.js 的 adm-zip 或 tar 命令
- UTF-8 无 BOM 编码写文件
- 每次修改更新版本号
- 构建前必须删除 dist 目录
